# multi-viewer

Multiple synchronized OpenSeadragon viewers

Renders colorized segmentations, heatmaps, etc.

The code uses HTML5, JavaScript ES6, CSS3, the npm package manager, Grunt automation...

## Install & Build

OS X & Linux:

```sh
npm install; grunt
```
